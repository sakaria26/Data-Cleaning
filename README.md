# Data-Cleaning
**Obtain a noisy data set and perform Data Cleaning using Data Reduction and Data Transformation**

## Data Reduction
**Use two(2) of the following methods to reduce the data set**

- Dimensionality Reduction
- Numerosity Reduction
- Data Compression

## Data Transformation
**Use two(2) of the following methods to reduce the data set**

- Smoothing
- Aggregation
- Discretization
- Normalization
